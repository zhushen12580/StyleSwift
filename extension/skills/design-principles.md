# 设计原则

> 内置知识：视觉设计的核心原则与最佳实践

## 一、对比度 (Contrast)

### 核心概念
对比度创造视觉层级和焦点，通过差异吸引注意力。

### 实践要点

1. **文字对比度**
   - 正文文字与背景对比度 ≥ 4.5:1 (WCAG AA)
   - 大标题对比度 ≥ 3:1
   - 使用工具检查：Chrome DevTools > Accessibility

2. **尺寸对比**
   - 标题字号是正文的 1.5-2 倍
   - 重要按钮比普通按钮大 20%

3. **颜色对比**
   - 主要 CTA 使用品牌强调色
   - 次要元素使用中性色
   - 避免过度使用对比色

4. **权重对比**
   - 标题使用 600-700 字重
   - 正文使用 400 字重
   - 强调文字使用 500 字重

```css
/* 对比度示例 */
h1 { font-size: 2em; font-weight: 700; }
p { font-size: 1em; font-weight: 400; }
strong { font-weight: 600; }
```

## 二、层级 (Hierarchy)

### 核心概念
层级引导用户视线，从最重要到次要信息依次呈现。

### 实践要点

1. **F型阅读模式**
   - 重要信息放在左上和顶部
   - 次要信息从左到右展开
   - 细节信息放在下方

2. **视觉权重排序**
   - 第一层级：标题、主按钮
   - 第二层级：副标题、图片
   - 第三层级：正文、辅助信息

3. **间距建立层级**
   - 组内间距 < 组间间距
   - 相关元素靠近，无关元素远离

```css
/* 层级示例 */
.header { margin-bottom: 32px; }  /* 大间距分隔主要区块 */
.section { margin-bottom: 24px; }
.paragraph { margin-bottom: 16px; }  /* 小间距分隔段落 */
```

## 三、留白 (Whitespace)

### 核心概念
留白是设计元素，不是空白。它创造呼吸感和高端感。

### 实践要点

1. **页面边距**
   - 桌面端至少 40-60px
   - 移动端至少 20-24px
   - 内容不要贴边

2. **元素间距**
   - 相关元素：8-16px
   - 模块分隔：24-32px
   - 章节分隔：48-64px

3. **内部留白**
   - 按钮内边距：12px 垂直 + 24px 水平
   - 卡片内边距：24px
   - 输入框内边距：12px

```css
/* 留白示例 */
.page { padding: 40px; }
.card { padding: 24px; margin-bottom: 32px; }
.button { padding: 12px 24px; }
```

## 四、一致性 (Consistency)

### 核心概念
一致性建立用户信任，降低认知负担。

### 实践要点

1. **颜色一致性**
   - 限制主色在 3-5 种
   - 建立色板：主色、辅色、强调色、中性色

2. **排版一致性**
   - 限制字号在 4-6 种
   - 建立排版阶梯：h1, h2, h3, body, small

3. **组件一致性**
   - 相同功能使用相同样式
   - 所有按钮圆角统一
   - 所有卡片阴影统一

```css
/* 一致性示例 - 设计系统变量 */
:root {
  --color-primary: #0066cc;
  --color-secondary: #6c757d;
  --spacing-sm: 8px;
  --spacing-md: 16px;
  --spacing-lg: 24px;
  --radius: 8px;
}
```

## 五、响应式 (Responsive)

### 核心概念
适配不同屏幕尺寸，确保所有设备上的可用性。

### 实践要点

1. **断点参考**
   - 移动端：320px - 480px
   - 平板：481px - 1024px
   - 桌面：1025px+

2. **流式布局**
   - 使用百分比或 vw
   - 避免固定宽度

3. **移动优先**
   - 从小屏开始设计
   - 渐进增强到桌面端

## 六、可访问性 (Accessibility)

### 核心概念
确保所有用户都能使用，包括视觉障碍用户。

### 实践要点

1. **颜色不是唯一标识**
   - 错误提示加图标，不只是红字
   - 链接加下划线，不只是变色

2. **足够的点击区域**
   - 按钮/链接至少 44x44px
   - 相邻可点击元素间距足够

3. **焦点可见**
   - 保留或自定义 focus 样式
   - 不要移除 outline

```css
/* 可访问性示例 */
button:focus {
  outline: 2px solid #0066cc;
  outline-offset: 2px;
}

a:focus-visible {
  outline: 2px solid currentColor;
}
```

## 总结检查清单

- [ ] 文字对比度 ≥ 4.5:1
- [ ] 有清晰的视觉层级
- [ ] 充足的留白
- [ ] 样式保持一致
- [ ] 响应式适配
- [ ] 可访问性测试通过
