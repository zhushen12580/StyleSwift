# 数字女娲 MVP 推广方案

> 基于 The Minimalist Entrepreneur 方法论制定
> 制定日期：2026-03-25

---

## 核心原则

**跳过"发布"。专注"销售"。**

"病毒式成功"是个神话。每一个看似一夜成名的产品背后都是数月甚至数年的努力。你的任务是一个一个地销售，从每次互动中学习，建立势头。

---

## 现状分析

### 优势
- 已上架Chrome商店，产品可用
- 开源项目，有技术背书
- 隐私优先定位差异化
- 有PPIO赞助支持

### 挑战
- 需要用户自备API Key（使用门槛）
- 浏览器扩展品类竞争激烈
- 用户需要先理解"为什么需要这个"

## 四、定价策略

### 当前状态
- 产品免费使用
- 用户需自备API Key

### 阶段规划

| 阶段 | 时机 | 策略 | 目标 |
|------|------|------|------|
| **第一阶段** | 现在 | 完全免费 | 获取前100个真实用户 |
| **第二阶段** | 有付费反馈后 | 增值服务收费 | 验证付费意愿 |
| **第三阶段** | 产品成熟后 | Freemium模式 | 规模化变现 |

### 可选收费方向
1. **API额度包**：用户不需要自己管理API Key
2. **高级风格技能库**：精选设计师风格模板
3. **团队版**：多人协作、风格共享
4. **企业版**：品牌定制、批量部署

### 需要验证的问题
- [ ] 用户是否愿意为"自带API额度"付费？
- [ ] 用户是否愿意为"高级风格技能库"付费？
- [ ] 企业用户是否有定制需求？
