概述

欢迎使用数字女娲 (StyleSwift)！我们高度重视您的隐私保护。本隐私政策旨在向您说明我们如何收集、使用、存储和保护您的个人信息。

核心原则：数字女娲采用"隐私优先"设计理念。您的所有数据均存储在本地设备上，我们不会将您的个人信息传输到任何远程服务器。

数字女娲是一款 Chrome 浏览器扩展，通过 AI 技术帮助您个性化网页样式。在使用我们的服务前，请仔细阅读本隐私政策。

我们收集的信息

2.1 您主动提供的信息

API 配置信息 本地存储
  • API Key：您输入的 AI 服务 API 密钥（如 OpenAI、Anthropic 等）
  • API 地址：自定义的 API 端点 URL
  • 模型名称：您选择使用的 AI 模型标识

这些信息仅存储在您的本地浏览器中，用于与您选择的 AI 服务提供商进行通信。我们不会收集、存储或传输您的 API 密钥到我们的服务器。

2.2 使用过程中产生的数据

数据类型 说明 存储位置
会话数据 您与 AI 的对话历史、应用的样式记录 Chrome Storage + IndexedDB
用户画像 您的样式偏好设置 Chrome Storage
风格技能 您保存的自定义样式模板 Chrome Storage
域名信息 您使用扩展的网站域名（用于会话隔离） Chrome Storage

2.3 我们不收集的信息

  • 您的浏览历史记录
  • 您的个人身份信息（姓名、邮箱、电话等）
  • 您的密码或其他敏感凭据
  • 您的支付信息
  • 您的位置信息

2.4 扩展权限数据访问

本扩展需要以下浏览器权限才能正常运行。以下是每个权限对应的数据访问详情：

权限 用途 访问的数据
activeTab 获取当前标签页的访问权限，用于修改页面样式 当前标签页的DOM结构、元素样式、页面截图（仅当您主动发起请求时）
storage 在本地存储您的配置、会话和偏好设置 API密钥、用户配置、会话数据、用户画像、风格技能（仅存储在您的设备）
sidePanel 显示扩展的用户界面（侧边栏） 不访问任何页面数据，仅用于显示界面
scripting 向页面注入CSS样式代码，实现样式修改 仅向页面注入样式代码，不读取或收集页面内容
<all_urls> 在所有网站上运行，提供样式服务 当前网站的域名、DOM结构（用于分析和修改样式）

重要声明：上述权限仅在您主动使用扩展时才会被调用。我们不会在后台自动读取或收集您的任何数据。页面内容仅在您与AI对话时被分析和处理。

Chrome扩展权限说明

为提供核心功能，本扩展需要以下浏览器权限。我们承诺仅在必要时使用这些权限，且不会滥用或超范围使用。

最小权限原则 我们的承诺
  • 按需请求：仅在用户主动发起操作时才访问页面数据
  • 本地处理：所有数据处理均在本地完成，不上传到我们的服务器
  • 用户控制：您可随时在Chrome扩展管理中撤销这些权限

权限详细说明

activeTab

用途：获取当前活跃标签页的临时访问权限，用于分析和修改页面样式。

数据范围：仅限当前激活的标签页，不包含其他标签页或后台页面。

触发时机：仅在您点击扩展图标或发送消息时触发。

storage

用途：使用Chrome本地存储API保存您的配置、会话历史和偏好设置。

数据范围：所有存储数据均保存在您的本地浏览器中，不与任何远程服务器同步。

存储内容：

  • API密钥（经过混淆处理）
  • 用户配置（模型选择、API地址等）
  • 会话数据（对话历史、应用的样式）
  • 用户画像（您的样式偏好）
  • 风格技能（您保存的自定义样式）
scripting

用途：向网页注入自定义CSS样式代码，实现您要求的样式修改。

操作范围：仅注入样式代码（CSS），不执行任何数据收集或跟踪脚本。

安全性：注入的样式代码完全由本地生成，不包含任何外部脚本。

<all_urls> 主机权限

用途：允许扩展在所有网站上运行，为您访问的任何网页提供样式修改服务。

数据访问：仅当您在某个网站上主动使用扩展时，才会访问该网站的数据。

限制：我们不会在您未主动操作时自动访问任何网站数据。

信息的使用

我们收集的信息仅用于以下目的：

  1. 提供核心服务：使用您的 API 配置与 AI 服务通信，为您生成个性化网页样式
  2. 会话管理：保存您的对话历史和样式记录，支持撤销/重做操作
  3. 个性化体验：根据您的偏好设置优化样式生成结果
  4. 风格迁移：允许您在不同网站间复用已保存的样式
  5. 本地缓存：提高扩展响应速度，减少重复计算

数据处理流程

当您使用本扩展时，数据处理按以下流程进行：

步骤 操作 涉及数据 处理位置
1 用户输入指令 您输入的文本描述、页面截图（可选） 本地设备
2 分析页面结构 当前网页的DOM结构、元素样式 本地设备
3 构建AI请求 用户指令+页面上下文+用户画像 本地设备
4 发送至AI服务 加密的API请求（经HTTPS） 第三方AI服务
5 生成样式代码 AI返回的CSS样式指令 第三方AI服务
6 应用到网页 CSS样式被注入到当前页面 本地设备
7 保存会话记录 对话历史、应用的样式 本地存储

重要说明：所有数据处理均在您的本地设备上完成。我们不会将您的数据用于广告投放、用户画像分析或任何商业目的。

数据存储与安全

5.1 存储位置

您的所有数据均存储在本地设备上，使用 Chrome 浏览器提供的存储 API：

  • Chrome Storage API：存储会话索引、用户画像、风格技能等结构化数据
  • IndexedDB：存储对话历史等较大数据

5.2 数据安全措施

  • 本地隔离：数据按域名隔离存储，不同网站的会话互不干扰
  • 自动清理：超过 90 天未访问的会话会自动清理
  • 会话限制：每个域名最多保存 20 个会话，防止存储滥用
  • 手动清除：您可以随时在设置中清除所有历史数据

5.3 API 通信安全

当您使用 AI 功能时，您的请求会发送到您配置的 AI 服务提供商：

  • 通信通过 HTTPS 加密传输
  • 我们不会拦截、存储或分析您的 API 请求内容
  • 请参阅您使用的 AI 服务提供商的隐私政策（如 OpenAI、Anthropic 等）

信息共享与第三方

我们不会出售、出租或以其他方式商业化您的个人信息。

6.1 数据共享概述

数字女娲采用"隐私优先"架构，这意味着：

  • 无中心服务器：我们不运营任何收集用户数据的服务器
  • 无跟踪代码：扩展不包含任何分析或跟踪代码
  • 本地优先：您的所有敏感数据（API密钥、对话历史）仅存储在本地

6.2 第三方AI服务提供商

当您使用AI功能时，以下数据会发送给您选择的AI服务提供商：

发送的数据类型
  • 您的指令：您输入的自然语言描述（例如："把标题改成红色"）
  • 页面上下文：当前网页的DOM结构摘要（用于AI理解页面结构）
  • 用户画像：您的样式偏好设置（用于个性化生成）
  • 截图（可选）：如果您上传了参考图片，图片会一并发送

除上述AI服务通信外，我们仅在以下情况下可能披露信息：

您的权利

作为用户，您享有以下权利：

7.1 访问权

您可以随时在扩展设置中查看存储的所有数据，包括会话历史、用户画像和风格技能。

7.2 删除权

您可以通过以下方式删除数据：

  • 删除单个会话：在会话列表中删除特定会话
  • 清除历史数据：在设置中点击"清除历史"按钮
  • 卸载扩展：卸载扩展将删除所有相关数据
7.3 修改权

您可以随时修改您的 API 配置、用户画像和风格技能设置。

7.4 数据可携带权

您可以导出保存的风格技能，在其他设备或浏览器中使用。

7.5 撤回同意权

您可以随时停止使用本扩展，Chrome浏览器会自动删除扩展存储的所有数据。

儿童隐私

数字女娲并非针对 13 岁以下的儿童设计。我们不会故意收集 13 岁以下儿童的个人信息。如果我们发现误收集了儿童的个人信息，会立即采取措施删除。

如果您是家长或监护人，发现您的孩子未经您同意使用了本扩展，请联系我们，我们将协助处理。

隐私政策的变更

我们可能会不时更新本隐私政策。当我们对隐私政策进行重大变更时，我们会通过以下方式通知您：

  • 在扩展更新日志中说明
  • 在 GitHub 仓库发布更新公告
  • 更新本页面的"最后更新日期"

我们建议您定期查阅本隐私政策，以了解我们如何保护您的信息。继续使用本扩展即表示您同意受更新后的隐私政策约束。
